### Notifications Hindi Lumalabas
1. Check kung naka-login
2. Check internet connection
3. Run test script: `python test_all_notifications.py`
4. Check backend logs

### Chat Hindi Nag-uupdate
1. Check kung naka-connect sa SocketIO
2. Check internet connection
3. Run test script: `python test_chat_realtime.py`
4. Restart app

### Typing Indicator Hindi Lumalabas
- Normal lang, lumalabas lang pag nag-type ng 2+ seconds
- Nawawala after 2 seconds ng walang type

### Unread Count Mali
- Refresh ang conversation list
- Mark as read ang messages
- Check kung tama ang database
