================================================================================
  COMMON ISSUES & QUICK FIXES
================================================================================

ISSUE: "Module not found"
FIX:   pip install -r requirements_fixed.txt --force-reinstall

ISSUE: "Port 5000 already in use"
FIX:   Edit run.py, change port=5000 to port=5001

ISSUE: "Database locked"
FIX:   Close all apps, restart START.bat

ISSUE: "Upload folder doesn't exist"
FIX:   Create: mkdir static\uploads
