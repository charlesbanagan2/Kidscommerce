## Issue 1: Website Add to Cart Error ✅ FIXED
**Problem**: "Product is out of stock" error when adding items to cart
**Root Cause**: add_to_cart_ajax function was using SQLAlchemy ORM instead of Supabase REST API
**Solution**: 
- Replaced Product.query.get_or_404() with get_data('product', filters={'id': product_id})
- Replaced Cart.query operations with get_data(), update_data_by_id(), insert_data()
- Changed status check to product.get('status') == 'active'
**File Modified**: backend/app.py (lines ~7993-8060)

## Issue 2: Mobile App Timeout Errors ✅ FIXED
**Problem**: Mobile app showing "Request timeout. Server not responding" when fetching products/categories
**Root Cause**: Mobile app configured with wrong IP address (192.168.100.46 vs actual 192.168.1.20)
**Solution**: Updated backendHost in url_config.dart to '192.168.1.20'
**File Modified**: mobile_app/lib/config/url_config.dart
